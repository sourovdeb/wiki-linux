# Sources


This space is where new documents enter the system.

Use it when you want to add a note, article, or document and turn it into a readable wiki page.

## Raw Source Root

- `/home/sourov/wiki-sources`

## Workflow

- Put raw documents into `/home/sourov/wiki-sources`.
- Ingest them explicitly with `wiki ingest <path>`.
- The LLM writes persistent source pages into this wiki.
- Raw files remain unchanged.
