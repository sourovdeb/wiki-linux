# Start Here


This page is the easiest entry point for a new user.

## First Three Places To Click

- [[user/notes/my-day|Dashboard]] to see the todo list, session report, AI box, and system policy.
- [[system/overview|Machine overview]] to understand what this computer is tracking.
- [[sources/README|Sources]] to add articles, notes, or documents.
- [[_meta/recent|Recent changes]] to see what changed last.

## If You Want To...

- understand the computer: go to [[system/README|System]]
- add new material: go to [[sources/README|Sources]]
- write your own notes: go to [[user/notes/README|Notes]]
- track a project: go to [[user/projects/README|Projects]]
- run a health check: open [[_meta/reports/README|Reports]] or run `wiki lint`

## What The Main Areas Mean

- Sources: original files that stay unchanged.
- System: readable pages about your machine setup.
- Notes: short personal pages.
- Projects: longer work that spans multiple pages.
- Research: references and collected material.

## Recommended Viewer

Open the vault in Obsidian if you want the easiest clickable interface.

```bash
wiki open
```
